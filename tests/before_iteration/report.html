#Q1
极限计算中误用等价无穷小：求 lim_{x->0} (tan x - sin x) / x^3
Error Analysis: 在分子差式中直接对 tan x 和 sin x 分别用等价无穷小替换,得到 (x-x)/x^3 = 0,完全抹去了高阶无穷小项。等价无穷小替换的前提是只能用于乘除因子,差式中替换会丢失关键的高阶信息。正确做法是提取因子化差为积,或使用泰勒展开到三阶。

#Q2
对数列极限直接套用洛必达法则：求 lim_{n->inf} n*(a^(1/n)-1)
Error Analysis: 洛必达法则要求函数在某区间上可导且变量连续,而数列的变量 n 是离散的。直接把 n 当连续变量对离散下标求导,过程不严谨。虽然本题恰好猜对了答案,但在需要严格论证的题目中会出错。正确做法是换元 t=1/n 转化为函数极限,或使用泰勒展开。

#Q3
分段函数在分界点的可导性判断：f(x)=x^2*sin(1/x) (x!=0), f(0)=0

#Q4

#Q6

#Q7

Improvement Suggestions

建立条件检查的解题习惯

Q4（中值定理）、Q5（对称性积分）、Q6（反常积分收敛）的根本问题都是在使用定理/性质前没有逐条验证前提条件。建议在草稿纸上先用三行写出定理的三个前提条件,逐条打勾确认后再使用。

差式极限优先用泰勒展开

Q1（等价无穷小差式）是最典型的差式替换陷阱。建议养成习惯：遇到0/0型的差式极限,优先想到泰勒展开到足够高阶,而非盲目使用等价无穷小替换。

分段函数分界点回归定义

Q3（分段函数可导性）的错误源于直接用导数公式而非定义。建议在涉及分段函数分界点、绝对值函数尖点、max/min函数的可导性问题时,强制自己先写出导数定义的差商极限形式。

多解思维：卡住时换视角

Q10（微分方程）展示了一个重要策略：当从y的角度无法求解时,尝试把x当未知函数。类似地,在交换积分次序（Q8）、换元等场景中,要培养多角度思考的习惯。

端点检查不能忘记

Q9（幂级数收敛域）漏掉端点判断是幂级数最常见的失分模式。建议在求出收敛半径R后,在纸上强制写一行端点检查：x=R和x=-R各代入判断。

数列极限先换元再处理

Q2（数列极限洛必达）错误地直接对离散变量求导。建议形成固定流程：遇到数列极限题目,第一步写t=1/n换元转化为函数极限,再决定使用洛必达、泰勒还是等价替换。

多元微分的逻辑链要理清

Q7（多元可微条件）混淆了一元和多元的微分类似性。建议画一张逻辑关系图：偏导连续 -> 可微 -> (偏导存在+函数连续),每个箭头单向不可逆,配合反例加深记忆。

重积分换序的标准流程

Q8（交换积分次序）的画域再定限是最基本的动作要求。建议养成三步流程：由积分限写出区域不等式 -> 在坐标系中画出区域 -> 从新积分变量的角度重新写限。缺一不可。
