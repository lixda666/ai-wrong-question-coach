### #Q1 -- 

**题目：** 极限计算中误用等价无穷小：求 lim_{x->0} (tan x - sin x) / x^3

**错因分析：** 在分子差式中直接对 tan x 和 sin x 分别用等价无穷小替换,得到 (x-x)/x^3 = 0,完全抹去了高阶无穷小项。等价无穷小替换的前提是只能用于乘除因子,差式中替换会丢失关键的高阶信息。正确做法是提取因子化差为积,或使用泰勒展开到三阶。

**知识点：** 极限与等价无穷小

**错误类型：** 方法错误, 概念混淆

**同类练习题：**
1. 求 lim_{x->0} (sin x - arctan x) / x^3
   - 参考答案：用泰勒展开:sin x = x - x^3/6 + o(x^3), arctan x = x - x^3/3 + o(x^3),代入得极限为 1/6。
2. 求 lim_{x->0} (e^x - cos x) / x
   - 参考答案：差式不可分别替换。用泰勒展开 e^x = 1+x+x^2/2+o(x^2), cos x = 1-x^2/2+o(x^2),差为 x+o(x^2),极限为 1。

### #Q2 -- 

**题目：** 对数列极限直接套用洛必达法则：求 lim_{n->inf} n*(a^(1/n)-1)

**错因分析：** 洛必达法则要求函数在某区间上可导且变量连续,而数列的变量 n 是离散的。直接把 n 当连续变量对离散下标求导,过程不严谨。虽然本题恰好猜对了答案,但在需要严格论证的题目中会出错。正确做法是换元 t=1/n 转化为函数极限,或使用泰勒展开。

**知识点：** 数列极限与洛必达法则

**错误类型：** 方法错误

**同类练习题：**
1. 求 lim_{n->inf} n^2 * (1 - cos(1/n))
   - 参考答案：令 t=1/n->0,化为 lim_{t->0} (1-cos t)/t^2 = 1/2。
2. 求 lim_{n->inf} n * sin(1/n)
   - 参考答案：令 t=1/n->0,化为 lim_{t->0} sin t / t = 1。

### #Q3 -- 

**题目：** 分段函数在分界点的可导性判断：f(x)=x^2*sin(1/x) (x!=0), f(0)=0

**错因分析：** 在分界点处直接套用 x!=0 时的导数公式求极限,得出导函数在 x=0 处无极限的结论,错误推断不可导。分段函数在分界点的可导性必须回到导数定义用差商极限来判断,不能直接使用非分界点处的导函数表达式。导函数在 x=0 处不连续与 f(x) 在 x=0 处可导是两回事。

**知识点：** 导数定义与分段函数可导性

**错误类型：** 方法错误, 概念混淆

**同类练习题：**
1. f(x)=x^3*cos(1/x) (x!=0), f(0)=0,判断在 x=0 是否可导？
   - 参考答案：按定义 f'(0) = lim_{x->0} x^2*cos(1/x) = 0（无穷小乘有界）,可导。
2. f(x)=|x|,判断在 x=0 处是否可导？
   - 参考答案：左右导数分别为 1 和 -1,不等,不可导。

### #Q4 -- 

**题目：** 中值定理条件未验证就使用：f(x)=sqrt(x) 在 [0,1] 上用罗尔定理

**错因分析：** 没有逐条验证罗尔定理的三个条件就直接套用。本题 f(0)=0 != f(1)=1（端点值不相等）,且 f(x) 在 x=0 处不存在右导数。两个关键条件都不满足,罗尔定理不适用。实际上 f'(x)=1/(2*sqrt(x))>0,不存在导数为零的点。

**知识点：** 中值定理（罗尔定理）

**错误类型：** 方法错误, 概念混淆

**同类练习题：**
1. f(x)=x^2-2x 在 [0,2] 上,验证罗尔定理是否适用？
   - 参考答案：f 连续可导,f(0)=0=f(2),条件满足。f'(x)=2x-2=0 得 xi=1。
2. f(x)=|x-1| 在 [0,2] 上,罗尔定理是否适用？
   - 参考答案：在 x=1 处不可导（尖点）,罗尔定理不适用。

### #Q5 -- 

**题目：** 定积分中奇偶对称性误用：计算 int_{-1}^{2} x^3 dx

**错因分析：** 只注意到被积函数 x^3 是奇函数,但没有检查积分区间是否关于原点对称。奇偶对称性要求区间为 [-a,a] 且函数具有对应的奇偶性,两个条件缺一不可。本题区间 [-1,2] 不对称,不能直接套用对称性。

**知识点：** 定积分对称性

**错误类型：** 审题不清, 方法错误

**同类练习题：**
1. 计算 int_{-2}^{2} x^5 dx
   - 参考答案：区间对称 + 奇函数,积分为 0。
2. 计算 int_{-3}^{1} x^2 dx
   - 参考答案：区间不对称,直接计算得 28/3。

### #Q6 -- 

**题目：** 反常积分收敛性判断：int_{1}^{inf} 1/x^p dx 的收敛性

**错因分析：** 将被积函数趋于零当作积分收敛的充分条件,这是一个根本性的概念混淆。趋于零只是收敛的必要条件,远非充分。当 p=1 时被积函数 1/x->0 但积分发散（对数增长）,说明趋于零的速度不够快。判断反常积分收敛必须使用比较判别法或 p-积分判别法等严格工具。

**知识点：** 反常积分收敛性判别

**错误类型：** 概念混淆

**同类练习题：**
1. 判断 int_{1}^{inf} ln x / x^2 dx 的收敛性
   - 参考答案：当 x 充分大时 ln x/x^2 <= 1/x^{1.5},而 int 1/x^{1.5} dx 收敛（p=1.5>1）,由比较判别法知收敛。
2. 判断 int_{0}^{1} 1/x^q dx 的收敛性（瑕积分）
   - 参考答案：q<1 收敛,q>=1 发散。

### #Q7 -- 

**题目：** 多元函数可微性条件混淆：偏导数都存在能否推出可微？

**错因分析：** 直接将一元函数可导等价于可微的结论套用到多元函数,忽略了多元函数中偏导数存在与可微之间的逻辑鸿沟。偏导数存在只保证沿 x 和 y 两个方向的变化率存在,而可微要求全增量能表示为 A*dx + B*dy + o(rho),比偏导存在强得多。偏导数连续才是可微的充分条件。

**知识点：** 多元函数可微性条件

**错误类型：** 概念混淆

**同类练习题：**
1. f(x,y)=xy/(x^2+y^2) ((x,y)!=(0,0)), f(0,0)=0,在(0,0)处偏导是否存在？是否可微？
   - 参考答案：偏导存在（均为0）,但 f 在 (0,0) 不连续,故不可微。
2. 偏导数连续是可微的必要条件吗？
   - 参考答案：不是。偏导数连续是可微的充分条件,非必要。存在可微但偏导数不连续的函数。

### #Q8 -- 

**题目：** 交换累次积分次序：int_{0}^{1} dx int_{x}^{1} f(x,y) dy

**错因分析：** 交换积分次序时没有先画出积分区域再重新确定边界,而是凭感觉照搬上下限。原区域是三角形 0<=x<=1, x<=y<=1,改为先对 x 积分时应先固定 y：0<=y<=1, 0<=x<=y。没有画域再定限的习惯是这类错误的根本原因。

**知识点：** 重积分交换次序

**错误类型：** 方法错误

**同类练习题：**
1. 交换 int_{0}^{2} dx int_{x^2}^{2x} f(x,y) dy 的积分次序
   - 参考答案：画图后,交换为 int_{0}^{4} dy int_{max(0,y/2)}^{sqrt(y)} f(x,y) dx,需分段。
2. 交换 int_{0}^{1} dy int_{0}^{y} f(x,y) dx 的积分次序
   - 参考答案：交换后:int_{0}^{1} dx int_{x}^{1} f(x,y) dy（即Q8原形式）。

### #Q9 -- 

**题目：** 幂级数收敛域端点处理：求 sum_{n=1}^{inf} x^n/n 的收敛域

**错因分析：** 收敛半径 R=1 只给出开区间 (-1,1) 内收敛的结论,端点的收敛性需要单独代入原级数判断。漏掉端点检查是幂级数题目中最常见的失分点。本题 x=1 代入得调和级数发散,x=-1 代入得交错调和级数收敛,因此收敛域是 [-1,1) 而非 (-1,1)。

**知识点：** 幂级数收敛域

**错误类型：** 知识盲区

**同类练习题：**
1. 求 sum_{n=1}^{inf} x^n/n^2 的收敛域
   - 参考答案：R=1。两端点均收敛,收敛域 [-1,1]。
2. 求 sum_{n=1}^{inf} n*x^n 的收敛域
   - 参考答案：R=1。两端点通项不趋于零均发散,收敛域 (-1,1)。

### #Q10 -- 

**题目：** 一阶微分方程类别混淆：dy/dx = y/(x+ln y)

**错因分析：** 按常规思路始终把 y 当成未知函数,分别尝试可分离变量和一阶线性方法均失败后陷入僵局。本题的关键洞察是：当以 y 为未知函数行不通时,把 x 视为 y 的函数后方程化为一阶线性 dx/dy = (x+ln y)/y = (1/y)*x + (ln y)/y。这种反客为主的策略是高数微分方程中常见的进阶技巧。

**知识点：** 一阶微分方程解法选择

**错误类型：** 方法错误, 概念混淆

**同类练习题：**
1. 解 dy/dx = 1/(x+y)
   - 参考答案：令 u=x+y,化为可分离变量 u/(u+1) du = dx。
2. 解 (y^2-6x)*dy/dx + 2y = 0
   - 参考答案：把 x 当未知函数,化为一阶线性,解得 x = C*y^3 + y^2/2。
