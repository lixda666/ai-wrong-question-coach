#Q001 Math
求 $\displaystyle\lim_{x\to 0}\frac{\tan x-\sin x}{x^3}$。
错因分析： 等价无穷小替换**只能在乘除因子中整体替换**。本题分子是"相减"的差式，直接替换会把高阶小量全部抹掉，导致结果错误。这里恰好 $x-x=0$，掩盖了真实极限。
极限与等价无穷小
方法错误概念混淆
同类练习题

1. 求 lim_{x→0} (sin x - arctan x) / x³
参考答案：用泰勒展开: sin x = x - x³/6 + o(x³), arctan x = x - x³/3 + o(x³), 差值 = x³/6 + o(x³), 极限为 1/6。

2. 求 lim_{x→0} (eˣ - cos x) / x
参考答案：差式不可分别替换。eˣ ≈ 1+x+x²/2, cos x ≈ 1-x²/2, 差为 x + o(x), 极限为 1。

#Q002 Math
设 $a_n=n(\sqrt[n]{a}-1)\,(a>0)$，求 $\displaystyle\lim_{n\to\infty}a_n$。
错因分析： 洛必达法则要求函数在某区间上**可导且为连续变量**。数列 $n$ 是离散的，不能直接对离散下标求导。本题"蒙对"了结果，但过程不严谨，遇到真正需要这一步的题目就会出错。
数列极限与洛必达法则
方法错误
同类练习题

1. 求 lim_{n→∞} n² · (1 - cos(1/n))
参考答案：令 t = 1/n → 0, 化为 lim_{t→0} (1-cos t)/t² = 1/2。

2. 求 lim_{n→∞} n · sin(1/n)
参考答案：令 t = 1/n → 0, 化为 lim_{t→0} sin t / t = 1。

#Q003 Math
设 $f(x)=\begin{cases}x^2\sin\frac1x,&x\neq 0\\0,&x=0\end{cases}$，判断 $f(x)$ 在 $x=0$ 处是否可导。
错因分析： 在分界点处，**必须先回到导数定义**计算，不能直接套用非分界点处的导函数表达式。上面那个 $f'(x)$ 在 $x\to 0$ 时确实发散，但它描述的是 $x\neq0$ 处的导数，与 $x=0$ 处是否可导无关。
导数定义与分段函数可导性
方法错误概念混淆
同类练习题

1. f(x) = x³·cos(1/x) (x≠0), f(0)=0, 判断在 x=0 处是否可导？
参考答案：按定义 f'(0) = lim_{x→0} x²·cos(1/x) = 0（无穷小乘有界）, 可导。

2. f(x) = |x|, 判断在 x=0 处是否可导？
参考答案：左右导数分别为 1 和 -1, 不等, 不可导。

#Q004 Math
证明存在 $\xi\in(0,1)$ 使 $f'(\xi)=0$，其中 $f(x)=\sqrt{x}\,(0\le x\le 1)$。
错因分析： 罗尔定理要求 $f(a)=f(b)$ 且 $f$ 在 $[a,b]$ 连续、$(a,b)$ 可导。本题 $f(0)=0\neq f(1)=1$，**端点值根本不相等**，罗尔定理不适用；且 $f$ 在 $x=0$ 处导数不存在（切线垂直）。结论本身也不成立（$f'(x)=\frac1{2\sqrt x}>0$）。
中值定理（罗尔定理）
方法错误概念混淆
同类练习题

1. f(x) = x² - 2x 在 [0,2] 上, 验证罗尔定理是否适用？
参考答案：f 连续可导, f(0)=0=f(2), 条件满足。f'(x)=2x-2=0 得 ξ=1。

2. f(x) = |x-1| 在 [0,2] 上, 罗尔定理是否适用？
参考答案：在 x=1 处不可导（尖点）, 罗尔定理不适用。

#Q005 Math
计算 $\displaystyle\int_{-1}^{2}x^3\,dx$。
错因分析： 奇偶函数在对称区间上的积分性质，要求**积分区间关于原点对称**（即 $[-a,a]$）。本题区间是 $[-1,2]$，并不对称，不能直接套用。
定积分对称性
审题不清方法错误
同类练习题

1. 计算 ∫₋₂² x⁵ dx
参考答案：区间对称 + 奇函数, 积分为 0。

2. 计算 ∫₋₃¹ x² dx
参考答案：区间不对称, 直接计算得 28/3。

#Q006 Math
判断 $\displaystyle\int_{1}^{+\infty}\frac{1}{x^p}\,dx$ 的收敛性。
错因分析： "被积函数趋于 0"**只是必要条件，远非充分条件**。趋于 0 的速度不够快，积分仍可能发散（如 $p=1$ 时 $\int_1^\infty \frac1x dx=\ln x\to\infty$）。
反常积分收敛性判别
概念混淆
同类练习题

1. 判断 ∫₁^∞ ln x / x² dx 的收敛性
参考答案：当 x 充分大时 ln x/x² ≤ 1/x^{1.5}, p=1.5>1, 由比较判别法知收敛。

2. 判断 ∫₀¹ 1/x^q dx 的收敛性（瑕积分）
参考答案：q<1 收敛, q≥1 发散。

#Q007 Math
已知 $z=f(x,y)$ 在 $(x_0,y_0)$ 处两个偏导数 $f_x,f_y$ 都存在，能否推出 $f$ 在该点可微？
错因分析： 一元函数中"可导 ⇔ 可微"，但多元函数中**偏导数存在既推不出连续，也推不出可微**。可微要求全增量能写成 $\Delta z=A\Delta x+B\Delta y+o(\rho)$，比"两个方向偏导存在"强得多。
多元函数可微性条件
概念混淆
同类练习题

1. f(x,y) = xy/(x²+y²) ((x,y)≠(0,0)), f(0,0)=0, 在(0,0)处偏导是否存在？是否可微？
参考答案：偏导存在（均为0）, 但 f 在 (0,0) 不连续, 故不可微。

2. 偏导数连续是可微的必要条件吗？
参考答案：不是。偏导数连续是可微的充分条件, 非必要。存在可微但偏导数不连续的函数。

#Q008 Math
交换 $\displaystyle\int_{0}^{1}dx\int_{x}^{1}f(x,y)\,dy$ 的积分次序。
错因分析： 交换次序必须先由原式画出积分区域 $D:\ 0\le x\le 1,\ x\le y\le 1$，再从 $y$ 的角度看 $x$ 的范围。原错解没有真正重新确定区域边界。
重积分交换次序
方法错误
同类练习题

1. 交换 ∫₀² dx ∫ₓ₂²ˣ f(x,y) dy 的积分次序
参考答案：画图后, 交换为 ∫₀⁴ dy ∫_{max(0,y/2)}^{√y} f(x,y) dx, 需分段。

2. 交换 ∫₀¹ dy ∫₀ʸ f(x,y) dx 的积分次序
参考答案：交换后: ∫₀¹ dx ∫ₓ¹ f(x,y) dy。

#Q009 Math
求幂级数 $\displaystyle\sum_{n=1}^{\infty}\frac{x^n}{n}$ 的收敛域。
错因分析： 收敛半径只给出**开区间** $(-R,R)$。**端点 $x=\pm R$ 必须单独代入判断**，不能想当然地排除或包含。
幂级数收敛域
知识盲区
同类练习题

1. 求 ∑_{n=1}^∞ x^n/n² 的收敛域
参考答案：R=1。两端点均收敛（p-级数 p=2>1）, 收敛域 [-1,1]。

2. 求 ∑_{n=1}^∞ n·x^n 的收敛域
参考答案：R=1。两端点通项不趋于零均发散, 收敛域 (-1,1)。

#Q010 Math
解微分方程 $\displaystyle\frac{dy}{dx}=\frac{y}{x+\ln y}$（$y>0$）。
错因分析： 本题从 $y$ 角度看不是线性、也不是可分离；但若**把 $x$ 视为 $y$ 的函数**，方程化为 $$\frac{dx}{dy}=\frac{x+\ln y}{y}=\frac1y x+\frac{\ln y}{y},$$ 这正是关于 $x$ 的**一阶线性微分方程**（未知函数是 $x(y)$）。
一阶微分方程解法选择
方法错误概念混淆
同类练习题

1. 解 dy/dx = 1/(x+y)
参考答案：令 u = x+y, 化为可分离变量 u/(u+1) du = dx。

2. 解 (y²-6x)·dy/dx + 2y = 0
参考答案：把 x 当未知函数, 化为一阶线性, 解得 x = C·y³ + y²/2。

改进建议

建立条件检查的解题习惯

Q004（中值定理）、Q005（对称性积分）、Q006（反常积分收敛）的根本问题都是在使用定理/性质前没有逐条验证前提条件。建议做题时先用几行写出定理的前提条件, 逐条打勾确认后再使用。

差式极限优先用泰勒展开

Q001（等价无穷小差式）是最典型的差式替换陷阱。建议养成习惯：遇到 0/0 型差式极限, 优先想到泰勒展开到足够高阶, 而非盲目使用等价无穷小替换。

分段函数分界点回归定义

Q003（分段函数可导性）的错误源于直接用导数公式而非定义。涉及分段函数分界点、绝对值函数尖点、max/min 函数的可导性问题时, 强制先写出导数定义的差商极限形式。

多解思维：卡住时换视角

Q010（微分方程）展示了一个重要策略：当从 y 的角度无法求解时, 尝试把 x 当未知函数。在交换积分次序（Q008）、换元等场景中, 要培养多角度思考的习惯。

端点检查不能忘记

Q009（幂级数收敛域）漏掉端点判断是幂级数最常见的失分模式。建议在求出收敛半径 R 后, 强制写一行端点检查：x=R 和 x=-R 各代入判断。

数列极限先换元再处理

Q002（数列极限洛必达）错误地直接对离散变量求导。建议形成固定流程：数列极限题目, 第一步写 t=1/n 换元转化为函数极限, 再决定用洛必达/泰勒/等价替换。

多元微分逻辑链要理清

Q007（多元可微条件）混淆了一元和多元的微分类似性。建议画逻辑关系图：偏导连续 → 可微 → (偏导存在+函数连续), 每个箭头单向不可逆, 配合反例加深记忆。

重积分换序的标准流程

Q008（交换积分次序）的画域再定限是最基本的动作要求。养成三步流程：由积分限写出区域不等式 → 在坐标系中画出区域 → 从新积分变量角度重写限。缺一不可。
